# Defragmentation Training School - NEUBIAS Academy & EOSC-Life 2022

![](Defragmentation_TS_2022_banner.png)


A training school for the new generation of bioimage analysts. Topics: workflow-based image analysis and new integrated methods for cloud computing applied to life sciences.

# Training School preparation #
#### Day 1: <br> <p>a) before the TS starts the trainees are pleased asked to register an account for https://usegalaxy.eu/login and optionally a GitHub account: https://github.com/ </p>
#### Day 2: <br> <p>a) before Day 2 the trainees need to have a working Google account (not a linked account that you manage with Google) to run Colab notebooks: https://colab.research.google.com/notebook . The Google account should have enough free space (~ several GB) on it.</p> <p>b) follow instructions from Dr. Christian Tischer to access BAND and run the script that installs the needed software on the remote desktop.</p><p>c) read the message from Dr. David Rousseau about slides and introduction to Google Colab for those who are not familiar yet with it.</p>
#### Day 3: <br> <p>a) it is required to have a Google Drive account and an instance of Fiji with the plugin deepImageJ.</p> <p>b) download CellProfiler here: https://cellprofiler.org/releases . Good if trainees get familiar with CP beforehand by e.g. watching this tutorial video: https://www.youtube.com/watch?v=QrzHQIiIDKM .</p>
#### Day 4: <br> <p>a) trainees can try the devbio-napari plugin collection before the lesson: https://github.com/haesleinhuepf/devbio-napari#installation.</p>
#### Day 5: <br> <p>a) Trainees are kindly asked to complete the presentation for the session "Work on your own data".</p><p>b) trainees have to register a GitHub account before the lesson: https://github.com/.</p><p>c) during the lesson trainees will be asked to register a Docker Hub account with name corresponding to the GitHub one: https://hub.docker.com/.</p>
